# Major Projects Collection

A collection of advanced full-stack projects showcasing different technologies and frameworks. Currently featuring a Job Portal built with modern web technologies.

## 🚀 Projects

### 1. Job Portal
A modern job search and recruitment platform built with React, TypeScript, and Tailwind CSS.

#### Features
- **User Authentication**
  - Login and SignUp functionality
  - Secure user sessions

- **Job Seeker Features**
  - Browse job listings
  - Apply for jobs
  - Build and manage resume
  - Track application history
  - View company profiles

- **Employer Features**
  - Post job openings
  - Review applications
  - Company profile management
  - Talent search

- **Additional Features**
  - Advanced search filters
  - Real-time notifications
  - Mobile responsive design
  - Profile customization

#### Tech Stack
- **Frontend**
  - React with TypeScript
  - Tailwind CSS for styling
  - Vite for build tooling
  - React Router for navigation

- **UI Components**
  - Custom designed components
  - Responsive layouts
  - Modern UI/UX principles

#### Project Structure
```
Jobportal/
├── frontend/
│   ├── src/
│   │   ├── assets/          # Static assets (images, icons)
│   │   ├── components/      # Reusable UI components
│   │   ├── pages/          # Page components
│   │   ├── Data/           # Mock data and types
│   │   └── utils/          # Utility functions
│   └── public/             # Public assets
```

## 🛠️ Setup and Installation

### Job Portal Setup
1. Navigate to the project directory:
   ```bash
   cd Jobportal/frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit:
   ```
   http://localhost:5173
   ```
